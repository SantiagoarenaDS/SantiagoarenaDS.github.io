# Santiago Arena — Portfolio

Portfolio personal estatico preparado para publicar con GitHub Pages.

## Archivos

- `index.html`: contenido principal de la pagina.
- `styles.css`: estilos, layout responsive y tema claro/oscuro.
- `script.js`: cambio de tema con persistencia en `localStorage`.
- `projects/`: paginas internas de proyectos y casos de estudio.
- `assets/projects/`: imagenes limpias usadas por la web.

## Material fuente

La carpeta `proyectos a incluir/` queda ignorada por Git porque contiene notebooks, PDFs de trabajo y material privado. Para publicar la pagina, usar los assets ya extraidos en `assets/projects/`.

## Publicacion en GitHub Pages

1. Crear un repositorio llamado `SantiagoarenaDS.github.io`.
2. Subir estos archivos a la rama principal.
3. En GitHub, ir a `Settings > Pages` y publicar desde la rama `main`.

La pagina quedara disponible en:

```text
https://SantiagoarenaDS.github.io/
```
